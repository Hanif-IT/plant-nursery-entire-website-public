# 🌿 Roghani Nursery - Full-Stack E-Commerce Platform

A complete online plant nursery business solution built with **PHP**, **MySQL**, and **HTML/CSS**.
Features a customer-facing store and a powerful admin panel for managing products, orders, and users.

## 🚀 Key Features
- **Customer Store:** Browse plants, filter by category, add to cart, and place orders.
- **Admin Dashboard:** Manage products (add/edit/delete), view pending/delivered orders, and track sales.
- **User Authentication:** Secure login/signup for both customers and administrators.
- **Order Management:** Real-time order tracking and status updates.
- **Responsive Design:** Works seamlessly on desktop and mobile devices.

- **Backend:** PHP (Procedural/OOP)
- **Database:** MySQL (via phpMyAdmin)
- **Frontend:** HTML5, CSS3, Bootstrap (optional)
- **Server:** XAMPP (Localhost)
- **Admin Panel:** Custom-built dashboard
